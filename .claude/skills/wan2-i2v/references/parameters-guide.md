# Parameters Guide

Wan I2V 動画生成パラメータの詳細ガイド。

## Resolution Settings

### 推奨解像度

| アスペクト比 | 解像度 | 用途 | VRAM目安 |
|-------------|--------|------|----------|
| 9:16（縦長） | 576×1024 | スマホ向け、ポートレート | 10-12GB |
| 16:9（横長） | 1024×576 | PC向け、ランドスケープ | 10-12GB |
| 9:16（低VRAM） | 480×840 | メモリ節約 | 6-8GB |
| 16:9（高品質） | 1280×720 | 高解像度出力 | 14-16GB |
| 1:1（正方形） | 768×768 | SNS向け | 10-12GB |

### 解像度選択のガイドライン

```
VRAM < 8GB  → 480×840 または 840×480
VRAM 8-12GB → 576×1024 または 1024×576
VRAM 12-16GB → 720×1280 または 1280×720
VRAM > 16GB → 任意（最大1080p程度）
```

### 解像度とフレーム数のトレードオフ

高解像度では生成可能なフレーム数が減少する：

| 解像度 | 最大フレーム数（12GB VRAM） |
|--------|---------------------------|
| 480×840 | ~121フレーム |
| 576×1024 | ~81フレーム |
| 720×1280 | ~49フレーム |

## Frame Count Settings

### フレーム数と動画長

| フレーム数 | 動画長（24fps） | 動画長（16fps） |
|-----------|----------------|----------------|
| 41 | ~1.7秒 | ~2.5秒 |
| 61 | ~2.5秒 | ~3.8秒 |
| 81 | ~3.4秒 | ~5.0秒 |
| 121 | ~5.0秒 | ~7.5秒 |

### フレーム数選択のヒント

- **初回テスト**: 41フレームで素早く確認
- **標準生成**: 81フレームが品質とスピードのバランス◎
- **長尺生成**: 121フレーム（VRAM注意）

## CFG (Classifier-Free Guidance)

### CFG値と出力特性

| CFG値 | 特性 | 用途 |
|-------|------|------|
| 2.0-3.5 | 最大限のクリエイティブ変化 | 実験的、アブストラクト |
| 3.5-4.5 | 自然で大きな動き | ダイナミックなシーン |
| 4.5-5.5 | バランスの取れた動き | 一般的な用途（推奨） |
| 5.5-6.5 | プロンプトに忠実 | 特定の動きを再現 |
| 6.5-7.5 | 強いプロンプト追従 | 動き少なめ、詳細重視 |

### CFG調整のベストプラクティス

```
動きが少ない → CFG を下げる（3.5-4.5）
プロンプトに従わない → CFG を上げる（6.0-7.0）
時間的不整合 → CFG 4.5-5.5 に戻す
```

## Sampling Steps

### Steps数と品質

| Steps | 生成速度 | 品質 | 推奨用途 |
|-------|----------|------|----------|
| 15 | 最速 | 低 | ラフプレビュー |
| 20 | 速い | 中 | テスト生成 |
| 25 | 標準 | 中高 | 一般用途（推奨） |
| 30 | やや遅い | 高 | 最終出力 |
| 40+ | 遅い | 高（収穫逓減） | 非推奨 |

### Steps調整のガイドライン

```
細部がぼやける → Steps を上げる（25-30）
生成が遅すぎる → Steps を下げる（20）
時間的不整合 → 過度なSteps（40+）を避ける
```

## Seed Settings

### シード値の使い方

| 設定 | 動作 | 用途 |
|------|------|------|
| -1 | ランダム生成 | 探索・バリエーション |
| 固定値 | 再現可能 | 微調整、比較 |

### シード固定のベストプラクティス

1. 良い結果が出たらシード値をメモ
2. パラメータ調整時はシードを固定
3. 最終確認後、複数シードで品質チェック

## Negative Prompt

### 効果的なネガティブプロンプト

```
blurry, low quality, distorted, deformed, ugly,
bad anatomy, watermark, text, logo,
oversaturated, underexposed, noise
```

### カテゴリ別ネガティブプロンプト

**品質系:**
```
blurry, low quality, jpeg artifacts, pixelated, noise
```

**解剖学系（人物）:**
```
bad anatomy, deformed, disfigured, extra limbs, missing limbs
```

**技術系:**
```
watermark, text, logo, signature, frame, border
```

## Motion Strength（TI2V用）

Wan2.2 5B TI2V で使用するパラメータ。

| 値 | 効果 |
|----|------|
| 0.3-0.5 | 微妙な動き、静止画に近い |
| 0.5-0.7 | 自然な動き（推奨） |
| 0.7-0.9 | ダイナミックな動き |
| 0.9-1.0 | 最大の変化 |

## Denoise Strength

画像からの変化度合いを制御。

| 値 | 効果 |
|----|------|
| 0.5-0.6 | 元画像に近い |
| 0.6-0.7 | バランス（推奨） |
| 0.7-0.8 | 大きな変化 |
| 0.8-1.0 | ほぼ新規生成 |

## Parameter Presets

### プリセット：クイックテスト

```yaml
resolution: 480×840
frames: 41
cfg: 5.0
steps: 20
seed: -1
```

### プリセット：標準生成

```yaml
resolution: 576×1024
frames: 81
cfg: 5.5
steps: 25
seed: -1
negative: "blurry, low quality, distorted"
```

### プリセット：高品質

```yaml
resolution: 720×1280
frames: 81
cfg: 6.0
steps: 30
seed: (固定値)
negative: "blurry, low quality, distorted, deformed, watermark"
```

### プリセット：ダイナミックモーション

```yaml
resolution: 576×1024
frames: 81
cfg: 4.0
steps: 25
motion_strength: 0.8
```

## パラメータ組み合わせの注意点

### やってはいけない組み合わせ

| 組み合わせ | 問題 |
|-----------|------|
| 高解像度 + 多フレーム | OOM |
| 高CFG + 高Steps | 時間的不整合 |
| 低CFG + 低Motion | 動きなし |

### 推奨の組み合わせ

| 目的 | 推奨設定 |
|------|----------|
| 安定した生成 | CFG 5.0, Steps 25 |
| 動きを増やす | CFG 4.0, Motion 0.8 |
| 品質重視 | Steps 30, 固定シード |
